## ✨ About The Project

**DigiTools** is a sleek, responsive web application designed for creators and professionals. It features a seamless product discovery experience, dynamic pricing comparisons, and a robust cart management system. Built with the latest web technologies, it ensures high performance and a modern user interface.

---

## 🚀 Key Features

### 1. Advanced Cart Management & Real-time Sync

Experience a robust shopping flow with a dynamic cart system. It handles item addition/removal with instant state updates, prevents duplicate entries, and calculates total costs using **ES6 .reduce()**.

### 2. High-Performance Data Handling with React 19

The platform leverages the cutting-edge **`use` hook** from React 19 to manage asynchronous data fetching for pricing and product catalogs. This ensures a faster, non-blocking UI compared to traditional patterns.

### 3. Modern Logic & Interactive UI (JS ES6+)

The project implements **JavaScript ES6+** best practices like **Ternary Operators** for conditional rendering (e.g., toggling between Product and Cart views) and dynamic styling. High-order functions like **.map()**, **.filter()**, and **.reduce()** are used to manage data flow with precision.

---

## 🛠️ Tech Stack

- **Frontend:** React 19 ⚛️ (Utilizing the new `use` hook).
- **Core Logic:** JavaScript ES6+ 💡 (Arrow Functions, Template Literals, and Ternary Operators).
- **Styling:** TailwindCSS v4 🎨 (New engine with `bg-linear-to-*` utilities).
- **Component Library:** **DaisyUI** 🌼 (For clean, accessible, and themeable UI components).
- **State Management:** React **useState** and **Props Lifting** (For efficient global state flow).
- **Notifications:** React Toastify 🔔 (Success/Warning alerts for cart actions).
- **Icons:** **Font Awesome** (For high-quality vector visuals).
- **Build Tool:** **Vite** ⚡ (For an optimized and lightning-fast development experience).

## 📦 Installation & Setup

1.  **Clone the repository:**

    ```bash

    git clone https://github.com/salmansahed/DigiTools.git

    ```

2.  **Navigate to the project directory:**

    ```bash

    cd digitools

    ```

3.  **Install dependencies:**

    ```bash

    npm install

    ```

4.  **Run the development server:**

    ```bash

    npm run dev

    ```
